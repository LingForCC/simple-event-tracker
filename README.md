# 简单事件追踪微信小程序

一个简洁的微信小程序，用于记录和追踪某一天某个事件是否发生，类似习惯追踪但功能更简单。

## 功能特点

- 📅 **日历视图** - 直观的月历显示，快速查看每日记录状态
- ✅ **多次记录** - 支持同一天多次记录事件，自动添加时间戳
- 📊 **统计信息** - 实时显示当月的完成情况统计
- 🔄 **修改删除** - 支持修改和删除已有记录
- 💾 **本地存储** - 数据保存在本地，无需网络连接
- 🎨 **简洁界面** - 清爽的设计，专注核心功能

## 使用方法

### 1. 查看日历
- 打开小程序即可看到当前月份的日历
- 绿色数字表示该日期的记录数量
- 支持月份切换和快速返回今天
- 可以切换月份查看历史记录

### 2. 记录事件
- 点击任意日期进入记录页面
- 点击"添加记录"按钮记录当前时间点的事件
- 支持同一天多次记录，每次记录都带有精确时间戳
- 记录后会显示在当天的时间列表中

### 3. 管理记录
- 点击已有记录的日期查看详细列表
- 可以删除单条记录或清空当天所有记录
- 支持下拉刷新更新数据

### 4. 查看统计
- 日历下方显示统计信息
- 当月活跃天数（有记录的天数）
- 当月总记录次数
- 全部时间总记录次数

## 技术特点

- **纯前端实现** - 无需后端服务器
- **本地存储** - 使用微信小程序的 `wx.setStorageSync` API 进行数据持久化
- **响应式设计** - 适配不同屏幕尺寸
- **实时更新** - 支持下拉刷新和数据实时同步
- **时间戳记录** - 精确到秒的记录时间
- **月份导航** - 完整的月份切换和今天定位功能

4. 点击编译即可预览

## 数据存储

数据使用微信小程序的本地存储功能：
- 存储键名：`eventRecords`
- 数据格式：
```javascript
{
  "2024-01-15": [
    { time: "09:30:25", timestamp: 1705315825000 },
    { time: "14:22:10", timestamp: 1705334130000 }
  ]
}
```
- 每个日期可存储多条记录，每条记录包含：
  - `time`: 格式化的时间字符串 (HH:MM:SS)
  - `timestamp`: Unix时间戳（毫秒）

## 自定义使用

这个小程序可以用于追踪各种日常事件：
- 运动健身（每次运动的时间点）
- 学习工作（每次专注学习的开始时间）
- 饮食记录（每次用餐的时间）
- 症状监测（症状出现的时间点）
- 习惯养成（每次完成习惯的时间）
- 情绪记录（情绪变化的时间点）
- 药物服用（每次服药的时间）
- 任何需要记录具体发生时间的事件

## 注意事项

- 数据保存在本地设备上
- 卸载小程序会丢失所有数据
- 建议定期备份重要记录（可通过清空数据功能截图备份）
- 支持下拉刷新更新数据
- 同一天支持多次记录，适合需要记录时间点的场景

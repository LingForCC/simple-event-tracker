# 简单事件追踪微信小程序

一个简洁的微信小程序，用于记录和追踪某一天某个事件是否发生，类似习惯追踪但功能更简单。

## 功能特点

- 📅 **日历视图** - 直观的月历显示，快速查看每日记录状态
- ✅ **简单记录** - 点击日期即可记录事件是否发生
- 📊 **统计信息** - 实时显示当月的完成情况统计
- 🔄 **修改删除** - 支持修改和删除已有记录
- 💾 **本地存储** - 数据保存在本地，无需网络连接
- 🎨 **简洁界面** - 清爽的设计，专注核心功能

## 使用方法

### 1. 查看日历
- 打开小程序即可看到当前月份的日历
- 绿色 ✓ 表示事件已发生
- 红色 ✗ 表示事件未发生
- 可以切换月份查看历史记录

### 2. 记录事件
- 点击任意日期进入记录页面
- 选择"事件发生"或"事件未发生"
- 记录后会自动返回日历页面

### 3. 修改记录
- 点击已有记录的日期
- 可以修改记录状态或删除记录

### 4. 查看统计
- 日历下方显示当月统计信息
- 包括已完成、未完成和总记录数量

## 技术特点

- **纯前端实现** - 无需后端服务器
- **本地存储** - 使用微信小程序的 `wx.setStorage` API
- **响应式设计** - 适配不同屏幕尺寸
- **组件化开发** - 清晰的页面结构和样式分离

## 开发环境

1. 下载并安装微信开发者工具
2. 导入项目文件夹
3. 配置小程序 AppID（测试可使用测试号）
4. 点击编译即可预览

## 数据存储

数据使用微信小程序的本地存储功能：
- 存储键名：`eventRecords`
- 数据格式：`{ "YYYY-MM-DD": true/false }`
- `true` 表示事件发生，`false` 表示事件未发生

## 自定义使用

这个小程序可以用于追踪各种日常事件：
- 运动记录
- 学习打卡
- 饮食记录
- 睡眠质量
- 任何需要记录是否发生的事件

## 注意事项

- 数据保存在本地设备上
- 卸载小程序会丢失所有数据
- 建议定期备份重要记录
- 支持下拉刷新更新数据
